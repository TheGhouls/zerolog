# zerolog


## Install

you can install it using setuptools :


```
pip install -r requirements.txt
python setup.py install
```

## Running tests

Tests are written with pytest.

You can run test suite using 


```
python setup.py test
```

Or manualy using 


```
py.test -v tests/
```
